  - 文本格式
  - 字体设置

  - 字体加粗
  - 斜体字
  - 下划线
  - 删除线
  - 文字颜色
  - 单元格颜色
  - 单元格边框
  - 字体倾斜
  - 边框倾斜
  - 背景倾斜
  - 合并单元格
  - 水平对齐
  - 自动换行

## **初始化**
    <!DOCTYPE html>
    <html lang="en">
        <head>
            <meta charset="UTF-8">
            <title>X-XWorkSheet</title>
            <link href="XSheet.css" rel="stylesheet">
            <style>
                * {
                    padding: 0;
                    margin: 0;
                }
        
                html, body {
                    height: 100%;
                    display: block;
                }
            </style>
            <script src="XSheet.js" type="text/javascript"></script>
        </head>
        <body id="demo">
            <script>
              const dome = document.getElementById('demo');
              const xSheet = new XSheet(demo);
            </script>
        </body>
    </html>
